# Intermediate Machine Learning Projects

This folder contains intermediate-level machine learning projects.

These projects are stronger than beginner projects because they involve more realistic datasets, more preprocessing, categorical features, business or healthcare context, and better model interpretation.

The projects in this section are considered **Intermediate / Medium Level** because they require more than just basic model training. They need proper data understanding, feature handling, visualization, and explanation of results.

## Skills Covered

- Data cleaning and preprocessing
- Handling categorical variables
- Encoding techniques
- Feature selection
- Exploratory Data Analysis

- Model comparison
- Confusion matrix and accuracy metrics
- Regression error metrics

- Report writing with charts and observations

---

## Common Workflow Followed

Most projects in this section follow this structure:

```text
1. Import libraries
2. Load and inspect the dataset
3. Understand columns and target variable
4. Clean missing or inconsistent values
5. Encode categorical features
6. Perform EDA with charts
7. Select useful features
8. Split data into training and testing sets
9. Train one or more models
10. Compare model performance
11. Interpret important patterns
12. Write final conclusion
```

---

## Why These Projects Are Medium Level

These projects are more practical than basic projects because:

- They use real-world business or healthcare problems
- They include mixed data types
- They need more preprocessing
- The results need explanation, not only accuracy
- Some datasets may have class imbalance
- Model evaluation matters more because wrong predictions can affect decisions
